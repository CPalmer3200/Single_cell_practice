# Single cell RNA seq (scRNAseq) practice analysis ################################

This repository is dedicated to my showcase some of the practice analysis I have done
with scRNAseq data. 

All data used within this project is derived from 5 samples from the study:
https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE171524

This repository is intended as personal practice for GitHub user CPalmer3200.
